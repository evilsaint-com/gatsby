When dumping database
```
sudo -u postgres pg_dump database-name  > dump.sql
```

You get an error that looks like 

<figure class="figure text-center col-xs-12 col-sm-12 col-lg-12">
<img src="/static/532719ab-7409-4371-9037-be1ce0637a23.png" class="img-fluid" alt="perl: warning: Falling back to the standard locale ('c')">
  <figcaption class="figure-caption text-center fw-normal text-dark">perl: warning: Falling back to the standard locale ('c').</figcaption>
</figure>


The fix

```
dpkg-reconfigure locales
```

For completeness, we should also talk about importing a database 
```
su postgres
psql -U postgres  database-name  > dump.sql
```
