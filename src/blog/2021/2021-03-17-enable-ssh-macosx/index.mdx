<h5 class="step">Step 1</h5>

Click on the system preferences icon

<figure class="figure text-center col-xs-12 col-sm-12 col-lg-12">
  <img src="/static/b7fba445-87aa-450b-a4b9-5486714ffecd.png" class="img-fluid" alt="System Preferences Icon.">
  <figcaption class="figure-caption text-center fw-normal text-dark">System Preferences.</figcaption>
</figure>

<h5 class="step">Step 2</h5>

You will be presented with the system preferences panel. Click "Sharing".

<figure class="figure text-center col-xs-12 col-sm-12 col-lg-12">
  <img src="/static/cb0736ee-d53a-4348-b3a0-e5555417c15b.png" class="img-fluid" alt="System Preferences Control Panel.">
  <figcaption class="figure-caption text-center fw-normal text-dark">System Preferences Control Panel.</figcaption>
</figure>

<h5 class="step">Step 3</h5>

On the left-hand navigation menu, make sure "remote login" is checked.


<figure class="figure text-center col-xs-12 col-sm-12 col-lg-12">
  <img src="/static/a33bfe22-bcff-4c2e-9dd1-5510f66b436d.png" class="img-fluid" alt="Sharing Settings.">
  <figcaption class="figure-caption text-center fw-normal text-dark">Sharing Settings.</figcaption>
</figure>

<h5 class="step">Step 4</h5>

Click the plus icon as indicated by the arrow in the screenshot. From here we can add individual system users and provide them access. 

<figure class="figure text-center col-xs-12 col-sm-12 col-lg-12">
  <img src="/static/41f0deee-891a-4d12-9d10-398cf563f34d.png" class="img-fluid" alt="Adding Users To Share Settings Configuration.">
  <figcaption class="figure-caption text-center fw-normal text-dark">Adding Users To Share Settings Configuration.</figcaption>
</figure>

<h5 class="step">Step 5</h5>

Alternatively, if you wish to give access to everyone on the system you can select the radio control next to the label "Allow access for:" and select "All users" to enable remote login for all users

<figure class="figure text-center col-xs-12 col-sm-12 col-lg-12">
  <img src="/static/ffa6af59-b53a-4698-8c0a-a90d94eb409d.png" class="img-fluid" alt="Allow Access For All Users.">
  <figcaption class="figure-caption text-center fw-normal text-dark">Allow Access For All Users.</figcaption>
</figure>
